# 📋 Prompt · Atualizar Apresentação de Conteúdo Social

> Prompt **autocontido** pra colar numa conversa nova do Claude.ai (ou outra IA).
> Funciona com a apresentação atual anexada (PPTX, PDF, imagens dos slides ou link Google Slides).
> Saída: deck atualizado com as diretrizes do brandbook v.2026.09.

---

## Como usar

1. Abre conversa nova no Claude.ai (de preferência num projeto, com o brandbook v.2026.09 anexado).
2. Anexa a apresentação atual (PPTX/PDF/imagens).
3. Cola o prompt abaixo (do `═══` ao final).
4. Refina iterativamente — Claude vai gerar slide a slide; ajusta o que precisar.

   - "O que ficou" (lista bullet)
   - "Slides removidos" (se houver)
   - "Slides adicionados" (lista)

B) **DECK ATUALIZADO** — formato à sua escolha:
   - Se tiver acesso a tools de criação de pptx: gera o .pptx mantendo

## Como adaptar esse prompt em diferentes cenários

| Cenário | Adaptação |
|---------|-----------|
| Apresentação como **PPTX** anexado | Cola o prompt direto, anexa o arquivo. Claude lê com SKILL pptx. |
| Apresentação como **PDF** | Idem — Claude lê PDF nativamente. |
| Apresentação como **Google Slides** | Adicione no prompt: *"O deck atual está em [link Google Slides]. Use o conector Google Drive pra acessar."* (precisa de conector Drive/Slides ativo) |
| Apresentação como **conjunto de imagens** dos slides | Idem — Claude lê imagens. |
| **Não tem apresentação ainda**, quer criar do zero | Apaga a primeira tarefa do prompt (DIAGNÓSTICO INICIAL) e ajusta a tarefa 2 pra "**CRIE A ESTRUTURA INICIAL** a partir do zero, com base nas diretrizes acima. Foco: 18-25 slides cobrindo manifesto, audiência, tom, canais, calendário editorial, exemplos por pilar". |
| Quer apresentação em **inglês** | Adicione no fim: *"O output deve ser em inglês BR-flavored — gírias e termos gamer em inglês quando o original era em português ('drop', 'press start', 'lacrado/sealed', 'NF/invoice')."* |
